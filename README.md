# How to run
Clone the repository to your local dev environment 

Run 
`npm install`

Run 
`npm start `

# Reqs
* NodeJS 
* Ringtail
* ApiKey 
* ApiToken 
* free 8080 port 

# SpreadJSRingtailExention
Utilizies Ringtails UI extension framework to display a excel - like data grid, populated from Ringtail api
